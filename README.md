## 🚀 Guide d'Installation Avancé

### 🔧 Prérequis Système
```bash
Node.js >= 18.0.0
npm >= 9.0.0
Git >= 2.40.0
Compte Firebase (Plan Blaze recommandé)
```

### ⚡ Installation Rapide
```bash
# Cloner le projet premium
git clone https://github.com/votre-repo/medtours-maroc-premium.git
cd medtours-maroc-premium

# Installation des dépendances (72 packages)
npm install

# Configuration Firebase (voir guide détaillé)
cp firebase-config-example.js src/firebase.js
# Modifier src/firebase.js avec vos clés

# Démarrage serveur développement
npm run dev
# ➜ Local: http://localhost:5174/
```

### 🔥 Configuration Firebase Avancée
